RaidMitTracker

Track every raid mitigation and external cooldown in real time — without chat spam.

Cooldown data is shared silently between addon users via addon messages. Nothing appears in raid chat. The raid leader sees a live panel with class-colored bars counting down each spell.

Key Features

  • Silent communication — uses SendAddonMessage, invisible to non-addon users and raid chat.
  • Auto-report on group join — members automatically announce their available spells when entering a group.
  • Live cooldown bars — smooth, class-colored status bars update every frame.
  • Combat log backup — spell casts by members without the addon are still detected.
  • Resizable & draggable panel — position and size are saved between sessions.
  • Solo test mode — /rmt test loads dummy data so you can preview the UI without a group.
  • English / Korean UI — automatically switches based on your client locale (enUS / koKR).

Requirements

All raid members should install the addon for complete pre-combat roster coverage. Members without the addon can still be partially tracked via combat log detection, but their spell list won't be known until they cast.

Commands

CommandDescription
/rmtBroadcast a cooldown check request to all members (leader / assistant only).
/rmt showOpen the tracker panel (leader / assistant only).
/rmt resetClear all tracked roster data.
/rmt testLoad dummy data for solo UI testing.

How It Works

The raid leader sends a CHECK message via addon channel. Each member's addon replies with a HAVE message listing their available spell IDs. When a tracked spell is cast, a USED message is sent with the cast timestamp. The leader's addon calculates endTime = castTime + hardcoded CD and the bar counts down smoothly every frame.

Direct cooldown queries on other players are not available in the WoW API — this addon works around that limitation using self-reporting and combat log event detection.

Limitations

  • Requires the addon installed on all members for full pre-combat roster visibility.
  • Talent-reduced cooldowns are not currently detected (base CD values are used).

RaidMitTracker (한국어)

공대 생존기(공생기) 및 외부생존기 쿨타임 실시간 추적 애드온 — 공대 채팅에 아무것도 출력되지 않습니다.

공대원끼리 애드온 메시지로 쿨타임 정보를 조용히 공유합니다. 공대장 화면에는 직업 색상의 쿨타임 바가 실시간으로 업데이트됩니다.

주요 기능

  • 채팅 스팸 없음 — SendAddonMessage 사용, 채팅창에 전혀 표시되지 않음.
  • 자동 보고 — 공대 합류 시 본인 보유 스킬 자동 전송.
  • 실시간 쿨타임 바 — 매 프레임 업데이트, 직업 색상 적용.
  • 전투 로그 백업 — 애드온 미설치 공대원의 스킬 사용도 감지.
  • 리사이즈 & 드래그 — 위치/크기 자동 저장.
  • 솔로 테스트 모드 — /rmt test 로 혼자서도 UI 확인 가능.
  • 영어 / 한국어 자동 전환 — 클라이언트 언어 설정 기준.

명령어

명령어설명

주의 사항

  • 전체 공대원 설치 권장 — 미설치자는 전투 로그로 부분 감지만 가능.
  • 쿨타임 감소 특성은 현재 미지원 (기본 쿨타임 사용).
